Privacy & Hosting

This website is a private presence. To protect the privacy of visitors, fonts are hosted locally. The website is hosted via GitHub Pages. For technical reasons, IP addresses are transmitted to GitHub, Tailwind CSS, and Lucide Icons when loading the site, but are not actively stored by us.

Minecraft Server Data

When joining the "Craftify Mc" server, technical data (IP address, Minecraft UUID, username, chat logs, and game statistics) is processed. This is strictly necessary to ensure operation, prevent attacks, and enforce server rules. This data is not shared with third parties.

External Services & Your Rights

We embed YouTube videos and use external APIs (mcstatus.io) to display the live server status. These services may process your IP address.

You have the right to information, correction, and deletion of your data. Complaints can be directed to the Austrian Data Protection Authority.
