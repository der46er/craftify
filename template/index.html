How to use in OBS

  1. Click on your preferred color above and copy the URL of the opened page.
  2. In OBS, add a new source and select "Browser" (Browserquelle).
  3. Paste the copied URL into the "URL" field.
  4. Set Width to 800 and Height to 200, then click OK. You're done!
